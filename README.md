## Horse Mob for [Minetest](http://minetest.net/)

---
### Description:

A ridable horse forked from [KPGMobs][kpgmobs] originally by PilzAdam & KrupnovPavel. It has been ported to use the [Creatures MOB-Framework API][git.creatures].
